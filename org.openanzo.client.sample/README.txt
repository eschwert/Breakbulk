The project contains example java applications that demonstrate how to
using the Open Anzo java client's numerious features.

Ant and Maven build files are provided to demonstrate how Open Anzo can be included
in Ant builds and Maven projects.  Below are some simple instructions for running the examples
using both tools.

Building with Ant:

ant compile

Running with Ant:

ant run-AccessControl
ant run-CommandLinking
...


Building with Maven:

maven package

Running the jar's created by Maven:

cd target
java -cp openanzo-sample-3.0-SNAPSHOT.jar org.openanzo.sample.AccessControl
java -cp openanzo-sample-3.0-SNAPSHOT.jar org.openanzo.sample.CommandLinking
...
